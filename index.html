📖 Como Jogar

🎯 Objetivo

Forme palavras usando as letras disponíveis no deck. O nível avança quando você descobre a Palavra Mestra — aquela que usa todas as letras de uma vez.

▶ Como jogar

  • Toque nas letras do deck para formá-las no input
  • Toque em uma letra no input para devolvê-la ao deck
  • Clique em Enviar para tentar a palavra
  • Use 🔀 Embaralhar para reorganizar o deck

✅ Palavras aceitas

  • Mínimo de 3 letras
  • Verbos no infinitivo, singular e plural
  • C cedilha (Ç) e palavras com acento

❌ Não são aceitas

  • Nomes próprios ou gírias
  • Palavras de baixo calão
  • Verbos conjugados
  • Palavras já encontradas na rodada

⚡ Poderes

Use Letra Inicial para revelar a 1ª letra de todas as palavras, ou Letra Surpresa para uma letra aleatória. Cada poder é usado uma vez por rodada.

Tem certeza que deseja terminar partida?

As palavras não encontradas serão reveladas.
